Conclusión

En este documento HTML, hemos utilizado varias etiquetas para mostrar diferentes tipos de contenido:

  • h1, h2, h3, h4, h5, h6: Estas etiquetas se utilizan para definir encabezados, con h1 siendo el más importante y h6 el menos importante. Los atributos id, class y style permiten identificar, aplicar estilos y personalizar cada encabezado.
  • img: La etiqueta img se utiliza para insertar imágenes en la página. Los atributos src (fuente de la imagen), alt (texto alternativo) y width/height (dimensiones) son esenciales para definir la imagen y su presentación.
  • video: La etiqueta video permite incrustar videos en la página. Los atributos width y height definen las dimensiones del video, mientras que controls, autoplay y loop controlan la reproducción del video.
  • table: La etiqueta table se utiliza para crear tablas. Los atributos border, cellpadding y cellspacing ayudan a definir el estilo y el espaciado de la tabla y sus celdas.
  • textarea: La etiqueta textarea se utiliza para crear un área de texto donde los usuarios pueden ingresar texto. Los atributos rows y cols definen el tamaño del área de texto, mientras que placeholder proporciona un texto de marcador de posición y maxlength limita la cantidad de caracteres que se pueden ingresar.
